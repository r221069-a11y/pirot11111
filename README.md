# Python Bar

## 基本機能

「Python Bar」は、オーセンティックバーでの注文ハードルを下げ、カクテルの直感的な体験を提供するモバイルオーダーWebアプリです。<br>

* **🍸 体験で絞り込む3ステップ・レコメンド:** <br>
名前だけでは味が想像できないメニューも多いカクテル。「アルコールの有無」→「炭酸の有無」→「味わい（甘め/辛口）」の3つの質問に答えるだけで、今の気分にぴったりのカクテルを提案し、お酒に詳しくない方でも迷わず注文できる体験を提供します。<br>

* **🔥 リアルタイムな「みんなの注文」トレンド表示:** <br>
バックエンドで保持している直近の注文データを集計し、トップ画面にタグ形式で自動表示することで、次の一杯の注文を後押しします。<br>

* **🛒 スムーズで直感的なカート＆オーダー機能:** <br>
画面下部に常に追従するカートバーで合計金額と杯数を確認でき、商品の詳細モーダル（ポップアップ）から数量の調整と注文確定がシームレスに行えます。<br>

* **📱 バーの世界観を表現したUIと安心の年齢確認:** <br>
ダークトーンとゴールドを基調とした、スマホアプリのように操作できるリッチなデザインを採用し、アルコールメニュー選択時には年齢確認のアラートを表示する安全設計を組み込んでいます。<br>

## 使用技術

* **バックエンド (Backend):** Python / Flask <br>
* **フロントエンド (Frontend):** HTML / CSS <br>
* **データベース (Database):** インメモリデータベース（※デモ版の仕様上、Pythonの変数としてデータを保持しています）<br>
